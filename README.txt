This directory contains Rod's tools to process shell arguments

Building:

 - to build run make
 - to install run make install
 - to do both run make all

Installation:
 - installation is for gnu/linux systems

 - header files are copied to /usr/include/rodtools/shelltools
   include rodtools/shelltools/Processing.h in your file for Shell Argument Processing function declarations

 - libRShellTools.so copied to /usr/lib
   link with -lRShellTools
